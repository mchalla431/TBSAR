Current Application
Task Scheduler with RGB Red LED blink @ 100ms
Build: 7668 bytes Flash (23.6%), 584 bytes RAM (7.1%)
Toolchain: arm-none-eabi-gcc with -O2 optimization
Programming: lpc21isp @ 57600 baud
Architecture Flow:
1. SysTick timer generates 10ms interrupts
2. Scheduler calls tasks at 10/20/50/100ms periods using countdown timers
3. Tasks use ECU modules (Rgb, Sensors, Monitor, etc.)
4. ECU modules call MCU drivers (Dio, Adc, Uart, I2c, Spi, etc.)
5. MCU drivers access hardware registers to control peripherals
